# TrelloBoard

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install
```

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```

## Features

- Create Column
- Edit Column
- Delete Column
- Create Task
- Delete Task
- Drag Columns
- Drag Tasks through other tasks and between themselves
